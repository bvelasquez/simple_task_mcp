# Commands & Scripts

- Install deps: `npm install`
- Development server: `npm run dev` (tsx)
- Watch mode: `npm run watch`
- Build: `npm run build` (tsc output to `build/`)
- Start (production): `npm start` (runs `node build/index.js`)
- Alternative start scripts:
  - `npm run start:local`
  - `npm run start:production`
